## src/module 模块
### src/module/admin 后台模块
### src/module/default 前台模块
### src/module/admin api模块
使用如下命令创建模块：
```bash
nest g module module/api
```
使用如下命令创建控制器
```bash
nest g controller module/admin/login
```
